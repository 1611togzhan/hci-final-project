# AI-Assisted Data Wrangler & Visualizer

## What it does
This Streamlit app allows users to upload CSV, Excel, or JSON data, inspect data quality, apply cleaning and transformation operations, build visualizations, and export the cleaned dataset together with a transformation report.

## Features
- Upload CSV / XLSX / JSON
- Overview profiling
- Missing value handling
- Duplicate detection and removal
- Data type conversion
- Categorical cleaning and mapping
- Numeric outlier handling
- Scaling and normalization
- Column operations
- Validation rules
- Visualization builder
- Export cleaned data and transformation log
